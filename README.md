# 🚀 AI Career Copilot

AI Career Copilot is a full-stack web application that helps users analyze their resumes, identify skills, detect career domains, generate ATS-style scores, and receive personalized career guidance.

## 🌐 Live Demo

https://ai-career-copilot-e2qpnkxkm-agasthya-hanus-projects.vercel.app

Backend:
https://ai-career-copilot-backend-bo5z.onrender.com

## 📌 Features

### 🔐 Authentication

* User Signup
* User Login
* JWT-based Authentication

### 📄 Resume Analysis

* Upload PDF resumes
* Upload DOCX resumes
* Upload TXT resumes
* Automatic text extraction

## 🚀 Installation

### Clone Repository

git clone https://github.com/Agasthya-Hanu/ai-career-copilot.git

cd ai-career-copilot

### Backend Setup

cd backend

npm install

npm start

### Frontend Setup

cd client

npm install

npm run dev
